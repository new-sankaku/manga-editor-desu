# 020: モバイル/タブレット対応の改善

## 規模: L (大)

## 概要
タブレットでの漫画編集体験を改善し、タッチ操作に最適化する。

## 現状の問題
- タッチ操作でのキャンバス操作が不安定
- ピンチズームとキャンバスズームの競合
- サイドバーがモバイル画面を圧迫する
- ツールバーのボタンサイズがタッチ操作に不向き
- ペンタブレット（Apple Pencil等）の筆圧検知が未対応

## 修正内容
### Phase 1: タッチ操作の基本改善
- ピンチズーム/パンのキャンバスズームとの統合
- タッチイベントのデバウンス/スロットリング最適化
- ダブルタップでズームフィットなどのジェスチャー

### Phase 2: UIレイアウト
- サイドバーをドロワー式に変更（モバイル時）
- フローティングツールバーの導入
- コンテキストメニューのロングプレス対応

### Phase 3: ペン入力対応
- Pointer Events APIによるペン入力の識別
- 筆圧感知対応（PointerEvent.pressure）
- カスタムブラシへの筆圧パラメータ反映

## 影響範囲
- UI/UX全般
- キャンバス操作
- サイドバー/ツールバー
